## Comandos básicos
  
### Mostrar todas as bases de dados
  
`show dbs;`  
  
### Mudar banco de dados ou criar banco em memória caso não exista
  
`use nomeDoBanco;`  
  
### Verificar o banco de dados ativo no momento
  
`db;`  
  
### Remover banco de dados ativo no momento
  
`db.dropDatabase();`
